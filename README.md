# Analyse

## Compilation
* Pour compiler le programme :
	* Dans un terminal, déplacez vous dans la racine du projet.
	* lancez les commandes :
		1. cmake .
		2. make

## Exécution
* Pour lancer le programme :
	* Lancer la commande :
		* ./bin/Analyse
	* Appuyer une fois sur 'echap' pour quitter le programme.
